# 更新日志

## v1.1.5（2026-07-13）

### 多平台登录支持
- 新增酷狗音乐、汽水音乐、Apple Music、Spotify 登录支持
- 登录模态新增 6 个平台选项卡（网易云、QQ、酷狗、汽水、AM、SP）
- 点击"打开登录窗口"自动打开平台登录页面，登录后自动捕获 Cookie
- 各平台独立 Cookie 输入面板，支持手动导入

### 账号信息面板
- 用户模态显示所有已登录平台的完整信息
- 每个平台显示头像、用户名、VIP 状态
- 支持点击切换平台
- 添加各平台颜色标识（红/绿/蓝/粉/绿）

### 账号显示设置
- 设置面板新增"账号设置"折叠区域
- 显示已登录账号列表
- 支持选择显示哪些平台的歌曲（全选/全不选）
- 打开设置面板时自动刷新账号列表

### Cookie 持久化
- Cookie 存储到用户目录 `~/.mineradio-platforms/`
- 应用重启后自动加载已保存的登录状态
- 支持酷狗、汽水、Spotify 平台的登录状态持久化

### 登录检测
- 酷狗：检查 `KuGoo`, `KugooID`, `UserName`, `mid` 等 Cookie
- 汽水：检查 `sessionid`, `tt_webid`, `uid` 等 Cookie
- Spotify：检查 `sp_dc`, `sp_t` 等 Cookie

### Bug 修复
- 修复登录窗口立即关闭问题（移除 `parent` 属性，添加页面加载检测）
- 修复昵称显示为 Unicode 编码问题
- 修复设置面板登录状态不更新问题

## v1.1.4（2026-07-09）

### TTML 歌词与翻译支持
- 新增 Apple TTML 格式歌词解析器，支持逐词时间戳和翻译
- 歌词数据结构新增 `translation` 字段
- 支持网易云翻译歌词（tlyric）自动合并
- TTML 解析修复：空格处理（自动补全单词间缺失空格）

### 3D 歌词翻译显示
- 翻译歌词作为独立 3D mesh 渲染，带完整描边/发光效果
- 翻译位于主歌词下方，大小可调节
- 新增「翻译大小」滑块（设置 → 歌词 → 翻译）
- 翻译开关切换时实时重新渲染

### 桌面歌词翻译
- 桌面歌词新增翻译显示支持
- 新增「桌面歌词翻译」开关（设置 → 叠加效果）
- 翻译行样式：半透明白色，无描边

### 歌词源设置完善
- 歌词来源信息显示（平台标签 + 时间轴类型标签）
- 新增「自动搜索歌词」开关
- 新增「优先 TTML 歌词」开关

### 自定义歌词 TTML 支持
- 自定义歌词弹窗新增格式选择：LRC / TTML / 自动检测
- 自动检测模式：自动识别 TTML 和 LRC 格式

### 快捷键设置
- 新增「快捷键」设置区域（设置 → 快捷键）
- 支持配置：播放/暂停、上/下一首、歌词显示/隐藏、歌词放大/缩小、音量加/减、老板键
- 点击输入框后按下键盘组合键即可设置

### 音量滚轮调节
- 鼠标悬停在音量按钮上时，使用滚轮调节音量
- 向上滚动 +5%，向下滚动 -5%

### 歌词描边修复
- 描边层现在正确跟随摄像头方向（使用 gl_FrontFacing 翻转 UV）

### Bug 修复
- 修复 TTML 歌词空格缺失问题
- 修复翻译 mesh opacity 未更新导致翻译不可见
- 修复残留的 updateLyricTranslationOverlay 调用导致 JS 报错
